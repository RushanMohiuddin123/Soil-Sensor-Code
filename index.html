Soil Sensor Project

This project is a basic implementation of a soil moisture monitoring system using an Arduino Uno and a soil moisture sensor. It detects the water content in soil and displays the output as analog values via the serial monitor. The system can be used for applications like automated irrigation systems, plant health monitoring, or smart agriculture.
